API Overview


Location Request

http://[address]/api/location/[zipcode]
A Location Request returns the location's name, coordinates, population and ZIP code.
If you don't know the ZIP code, just simply log in and search your location.

Example Request

http://[address]/api/location/10002 

                {
                    "City": "NEW YORK", 
                    "Latitude": 40.71, 
                    "Longitude": -73.99, 
                    "Population": 81410, 
                    "Zipcode": 10002
                }

Notes

  • API responses consist of a UTF-8-encoded, JSON-formatted object
  • As I only have location data of United States for now,
    I'm working for getting South Korea's data as well.
  • Since my database has limited access, I politely ask you not to make excessive API requests.
  • The original database sourcefile is on my github project1 repostitory.
    I compressed the file to prevent from unwanted loading thousands of rows.
